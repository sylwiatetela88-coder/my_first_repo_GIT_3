+# Mój pierwszy projekt GIT

Sprawdziłam polecenie git log, pokazuje ono historię commitów w repozytorium , pokazuje ID commita, autora, datę a także opis zmian.
Git --oneline pokazuje wersję skróconą, każy commit jest pokazany w osobnej linijce, szybki przgląd historii.
Pozostałe flagi git log, które sprobowałam to: 
- -n 3, który wyświetla trzy ostatnie commity,
- --stat, pokazuje statystyki zmian,
- --relative-date pokazuje jaki czas temu był dodany dany commit,
- --graph to przedstawienie w formie graficznej drzewka commitów,
- -pretty=format:"%h %an %ar %s" własny format do wyświetlania .
Funkcje te pozwalają na śledzenie historii.
